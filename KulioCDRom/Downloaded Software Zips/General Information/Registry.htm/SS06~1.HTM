We all know how the Recycle Bin got its name. It's because Microsoft didn't want to face a look-and-feel lawsuit from Apple for appropriating the Macintosh's Trash Can concept. If you're ready to come clean and rename the Recycle Bin, you have two options--the hacker method and the Tweak UI method. Here's the hacker way:

    1. Make sure you have a pristine backup of the Windows Registry. We're not playing around here.
    2. Open regedit--select Start/Run, type regedit in the text box, and press Enter.
    3. Open HKEY_CLASSES_ROOT, and locate CLSID (the Class ID option). Scroll down to find the key: {645FF040-5081-101B-9F08-00AA002F954E}.
    4. Double-click (Default), and give the beast a new name.
    5. Click OK; then close regedit.
    6. Click once anywhere on the desktop; then press the F5 key to refresh so that the change will take effect.

And here's the Tweak UI way:

    1. Select Start/Settings/Control Panel, and launch Tweak UI.
    2. Under Desktop, right-click Recycle Bin, and select Rename.
    3. Type in whatever new name you want to give the item.
    4. Click OK.
