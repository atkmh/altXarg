4. save your key files
"I'll take Crucial Windows Files for $500, Alex."

"Windows advisers recommends that people back up these ten files using ERU, or the Emergency Recovery Utility."

"What are config.sys, autoexec.bat, win.ini, system.ini, protocol.ini, user.dat, system.dat, io.sys, command.com, and msdos.sys?"

"Correct! And that's today's Daily Double!"

Like Configuration Backup, the Emergency Recovery Utility ships on the Windows 95 CD-ROM, tucked away in an obscure folder (\Other\Misc\Eru, to be exact). Also like Configuration Backup, it's available for download if you don't have the CD-ROM on hand. But unlike Configuration Backup, ERU roots out a whole slew of Windows configuration and system files, backs them up, and can restore them if you experience a disk disaster of epic proportions.

An ERU backup is done by a Wizard--it's so simple, it doesn't even need a description here. The backup file isn't compressed, so it can take up more than a high-density floppy's worth of space. But they are bytes well spent (and if you do want to store them on a floppy, you can always use WinZip to crunch 'em).

The process of restoring the settings you've saved after disaster strikes has a couple of wrinkles, but it's far from difficult:

1. Start your system using an emergency boot disk (you have made one, right?), and when you see the words "Starting Windows 95," press the F8 key.

2. Select the Safe Mode Command Prompt Only option.

3. Use DOS's cd command to change to your backup directory (the syntax is cd c:\backup).

4. To run the restoration utility (erd.exe), type erd at the prompt, and restore whichever files have been corrupted.
