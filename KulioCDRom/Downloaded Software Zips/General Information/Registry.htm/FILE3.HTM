3. back up your Registry files
"I'll take Crucial Windows Files for $300, Alex."

"These files make up the Windows 95 Registry."

"Errrrr. Ummm...What are...uhhhh...?"

"Your time's up. Any takers?"

Though you need to back up your Registry files, you don't need to know what their names are. Just use the Configuration Backup utility to do it for you. This tool is tucked away on the Windows 95 installation CD-ROM in the \Other\Misc\Cfgback folder (or is available for download if you don't have the CD-ROM version).

Why do you have to back up the Registry data? Well, some people edit the Registry, and only fools rush in to this without a safety net. The Configuration Backup program enables you to set up alternative Registry configurations--up to nine of them. It compresses data as it goes, crunching more than a megabyte of Registry data down to around 200K.

Like any other files, those comprising the Registry are prone to cross-linking, bad sectors, and other file-corruption nightmares. As long as you can load Windows 95, Configuration Backup will be able to save the day. Of course, as disaster recovery procedures go, Configuration Backup is far from ideal. It saves only the Registry data, and it requires a functioning Windows to restore it (and hey, if the Registry's buggy, there ain't no Windows). The rest of the key files are handled by the Emergency Recovery Utility, described on the next page.
