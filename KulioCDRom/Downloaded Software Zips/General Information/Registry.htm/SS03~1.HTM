Ever noticed that CD-ROM icons change whenever you put in a new Windows 95-enabled disc? A CD-ROM designed to run under Windows 95 uses an Autorun file containing instructions for loading the disc's program, and Win 95 looks for this file on every drive--hard or CD-ROM. This trick is your key to changing the dull icon on your hard disk (or any other drive to which you have access).

So if you want to label your hard disk with a spiffy icon, first you've got to find a spiffy icon. They can be found in any file with the extensions .ico, .dll, or .exe. ICO files contain only a single icon; the other two types contain all manner of things and one or more icons. To browse for icons, do this:

    1. From Explorer or the desktop, find a Shortcut you've created.
    2. Right-click it, and select Properties.
    3. Click the Shortcut tab, and then click the Change Icon button.
    4. Click Browse, and trawl your hard disk for ICO, DLL, or EXE files.
    5. When you find an icon you like, note the filename; if there are multiple icons, note where in the lineup the desired one appears.
    6. Then open Notepad (in the Accessories group on the Start/Programs menu). In a new file, enter:

      [autorun]
      icon=filename.ext,number

    (where filename.ext is the name of the file containing the icon--such as moricons.dll--and number is the rank in which the icon appears; 0 is the first or only icon, 1 is the next, and so on).
     

    7. Save the file as autorun.inf in the root directory of your C: drive (or any other hard drive). You'll see your new icon whenever you look at the drive in Explorer or My Computer.
